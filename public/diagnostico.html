Errores en Consola

Abre la consola del navegador (F12) y verifica si hay errores.

Errores comunes:

  • Cannot read properties of null - Problema con datos nulos
  • Failed to load module - Error de importación
  • Firebase Error - Problema de conexión a Firebase

Pasos para solucionar

  1. Abre la consola del navegador (F12)
  2. Ve a la pestaña "Console"
  3. Busca mensajes de error en rojo
  4. Toma una captura de pantalla de los errores
